# HTML Resume Website

A beginner-friendly personal resume website developed using HTML as part of my web development learning journey. The project presents my educational background, skills, achievements, and personal information in a clean and organized layout.

## 🚀 Live Demo
https://hrkayy.github.io/Capstone-proj1/

## 📌 Features
- Personal profile summary
- Education section
- Technical and leadership skills
- Achievements and certifications
- Hobbies page
- Contact page
- Multi-page website using HTML links

## 🛠️ Technologies Used
- HTML5
